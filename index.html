For organizers: how to manage this schedule

The entire schedule is driven by one Google Sheet with a separate tab for each day. Edit the Sheet and this page updates the next time someone loads it. You never need to touch the code.

One-time setup

  1. Use the prepared Sheet, which has one tab per day (Tue Jul 21, Wed Jul 22, Thu Jul 23, Fri Jul 24). Every tab uses the same columns in this order: Start Time, End Time, Session Title, Requirement, Requirement Detail, Audience Group (lane), Target Audience (as written), Theme / Track, Format, Room / Location, Virtual Link, Presenter, Description, Also Offered At, Notes.
  2. Add one session per row on the matching day's tab.
  3. In the Sheet, go to File, Share, Publish to web. Publish each day tab separately as Comma-separated values (.csv) and copy each link.
  4. Open this page's HTML file, find the tabs list near the top of the script, and paste each tab's link into its csvUrl. Save. That is the only code change ever needed.

How to fill in each column

  • Start Time / End Time: e.g. 9:45 AM and 11:00 AM. Used for sorting, calendar export, and conflict warnings.
  • Session Title: the name shown on the card.
  • Requirement: Required or Choice. Requirement Detail carries the qualifier, e.g. One admin per building.
  • Audience Group (lane): the controlled value used for filtering, e.g. School-Based Administrators or All Administrators. Target Audience (as written) is your own wording, shown on the card.
  • Theme / Track: optional category. Each value gets its own color automatically.
  • Format: In-person or Virtual. For virtual sessions, add a Virtual Link; the card shows a join link instead of a room.
  • Room / Location, Presenter, Description, Notes: shown on the session. Also Offered At lists extra time slots for repeats.

Day-to-day changes

  • Room change or typo: edit the cell on that day's tab. The page shows it on the next load.
  • Add or cancel a session: add or delete the row on the matching day's tab.
  • Move a session to another day: cut the row and paste it onto the correct day's tab.

Tip: keep the column headers exactly as listed on every tab. Everything else is freely editable.
